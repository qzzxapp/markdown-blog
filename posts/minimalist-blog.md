## 什么是好的博客设计

好的博客设计应该让人**忘记设计本身**，专注于内容。

几个核心原则：

1. **排版优先** — 字体、行高、段间距决定阅读体验
2. **留白充足** — 让内容有呼吸的空间
3. **对比清晰** — 标题、正文、引用各有层次
4. **加载极快** — 静态文件，毫秒级响应

## Markdown 是最好的写作格式

```markdown
# 标题

正文内容，简单直接。

- 列表项 1
- 列表项 2

> 引用文字，用于强调
```

不需要富文本编辑器，不需要数据库，一个 `.md` 文件就是一篇文章。

## 这个博客的实现

这个博客使用了以下技术：

- **HTML5** 语义化标签
- **CSS3** 原生变量和 Grid 布局
- **marked.js** 客户端 Markdown 渲染
- **highlight.js** 代码语法高亮
- **零依赖后端** — 纯静态，可部署到任何地方
